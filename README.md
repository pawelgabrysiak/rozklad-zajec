# System Rozkładu Zajęć

Projekt akademicki systemu rozkładu zajęć, zrealizowany w ramach studiów z zakresu Informatyki Stosowanej.

Celem projektu było zaprojektowanie relacyjnej bazy danych oraz integracja jej z backendem aplikacji.

## Funkcjonalności
- Wyświetlanie rozkładu zajęć według grupy studenckiej
- Przypisywanie zajęć do prowadzących, sal i terminów
- Spójność danych dzięki kluczom obcym i ograniczeniom integralności

## Model danych
System oparty jest na relacyjnej bazie danych obejmującej m.in.:
- zajęcia
- prowadzących
- grupy studenckie
- sale
- terminy zajęć

## Technologie
- PHP
- SQL (MariaDB / MySQL)
- phpMyAdmin

## Cel edukacyjny
Projekt służył nauce:
- projektowania relacyjnych baz danych
- pracy z kluczami obcymi i indeksami
- integracji backendu z bazą danych
